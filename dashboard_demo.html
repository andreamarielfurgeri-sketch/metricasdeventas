Instalación y Ejecución

Pasos para ejecutar la aplicación completa:

  1. Instalar herramientas de desarrollo: xcode-select --install
  2. Instalar dependencias: pip3 install -r requirements.txt
  3. Ejecutar aplicación: streamlit run app.py
  4. Abrir navegador en: http://localhost:8501

Archivos del Proyecto:

  • app.py - Aplicación principal (25KB)
  • estilos.py - Configuración visual
  • filtros.py - Funciones de filtrado
  • calculos.py - Métricas y estadísticas
  • graficos.py - Generación de gráficos
  • requirements.txt - Dependencias
